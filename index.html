Vanilla Engine

A web UI Game engine to quickly check Sprite animations for your games.

Upload separate sprite sheets for Idle and Walk, then move with arrow keys/WASD.

Controls

  • Arrow keys or WASD: Move
  • P: Pause/Unpause
  • Click canvas to focus if keys don't respond.

Tip: Both sprite sheets should have the same frame size. Enter Frame W/H and FPS, then click Apply Settings.
